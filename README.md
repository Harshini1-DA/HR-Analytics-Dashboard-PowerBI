# HR Headcount Analysis Dashboard

## Project Overview
This is an interactive HR analytics dashboard developed using Microsoft Power BI. 
It analyzes employee headcount, gender distribution, department allocation, age groups, and country-wise workforce data. 
The dashboard helps HR teams make data-driven workforce decisions.

## Tools Used
- Microsoft Power BI
- Power Query
- DAX
- Data Modeling

## Key Features
- Total Employee Headcount
- Gender Distribution Analysis
- Age Group Segmentation
- Business Unit Distribution
- Country-wise Workforce Analysis
- Hiring Trend Visualization
